# Quizly Logo Guideline (v1)

## Core Concept
- キーワード: 「たのしい」「できた!」「学びが前に進む」
- 構成: `Quizly` ワードマーク + Qモチーフの学習バッジ
- 造形: 角丸・太線・軽いジャンプ感

## Colors
- Slate text: `#334155`
- Soft slate base: `#E2E8F0`
- Teal accent: `#0F766E`
- Zinc/slate outline: `#475569`

## Variants
- Horizontal: `public/brand/quizly-logo-horizontal-light.svg` / `...-dark.svg`
- Stacked: `public/brand/quizly-logo-stacked-light.svg` / `...-dark.svg`
- Mark only: `public/brand/quizly-mark-light.svg` / `...-dark.svg`
- Master editable vector: `public/brand/logo.svg`

## Exported PNG
- Mark: `512`, `256`, `128`, `64`
- Horizontal (reference): `512`, `256`, `128`, `64`

## Usage Rules
- 最小クリアスペース: ロゴ高さの `0.25x`
- 最小サイズ: 文字あり `120px`、マーク単体 `32px`
- 禁止事項:
  - 極端な変形
  - 影色/アウトライン色の改変
  - 低コントラスト背景上での使用
