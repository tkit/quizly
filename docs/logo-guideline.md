# Quizly Logo Guideline (v3)

## Core Concept
- キーワード: 「たのしい」「できた!」「学びが前に進む」
- 採用案: `proposal-a`（Qマークに4色、ワードマークは落ち着いた単色）
- 造形: 角丸・太線・軽いジャンプ感、4色は控えめアクセント

## Colors
- 国語 (Red): `#F43F5E`
- 算数 (Blue): `#3B82F6`
- 理科 (Orange): `#F97316`
- 社会 (Green): `#22C55E`
- Wordmark (Light): `#334155`
- Wordmark (Dark): `#E2E8F0`

## Variants
- Horizontal: `public/brand/quizly-logo-horizontal-light.svg` / `...-dark.svg`
- Stacked: `public/brand/quizly-logo-stacked-light.svg` / `...-dark.svg`
- Mark only: `public/brand/quizly-mark-light.svg` / `...-dark.svg`
- Approved proposal source: `public/brand/quizly-logo-proposal-a.svg`
- Master editable vector: `public/brand/logo.svg`

## Usage Rules
- 最小クリアスペース: ロゴ高さの `0.25x`
- 最小サイズ: 文字あり `120px`、マーク単体 `32px`
- 禁止事項:
  - 極端な変形
  - グラデーション表現の追加
  - ワードマーク文字の多色化
  - 低コントラスト背景上での使用
