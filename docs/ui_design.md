# 画面設計 (UI/UX)

本ドキュメントでは、学習アプリケーションの主要な画面構成とフローを定義します。
基本的にはモバイルファースト（スマートフォンやiPadで操作しやすいUI）で設計します。

## 画面一覧とフロー

1. **[ログイン画面]** `/(Home)`
   * ユーザー（子供）が自分のアカウントを選択する画面。
   * **遷移先:** [ダッシュボード画面]
2. **[ダッシュボード画面 (ジャンル選択)]** `/dashboard`
   * 学習の拠点となる画面。
   * **遷移先:** [問題設定画面]（ジャンル選択後）、[学習記録（履歴）画面]
3. **[モード・詳細設定画面]** `/setup?genre=math`
   * 選んだジャンルに対して、「何問解くか」「通常モードか復習（過去に間違えた問題）モードか」を設定する。
   * **遷移先:** [問題解答画面]
4. **[問題解答画面]** `/quiz?genre=math&mode=normal&count=10`
   * 実際に問題を解くメイン画面。
   * 1問解答するごとに正誤判定と解説をポップアップ（または画面下部）に表示し、「次の問題へ」進む。
   * 全問終了でリザルト画面へ遷移する。
   * **遷移先:** [リザルト（結果）画面]
5. **[リザルト（結果）画面]** `/result?session_id=123`
   * 完了したセッションのスコア（〇問中〇問正解）と、かかった時間などを表示。
   * スコアに応じた演出（紙吹雪など）でモチベーションを高める。
   * **遷移先:** [ダッシュボード画面]
6. **[学習履歴・成績画面]** `/history`
   * 今日、今週、トータルの学習回数などをグラフィカルに表示（プログレスバーなど）。

## 画面構成案（ワイヤーフレーム相当）

デザインシステムは **shadcn/ui** をベースとし、明るすぎないが子供が親しみやすいカラーパレット（明るいブルー、オレンジ、グリーン等のアクセントカラー）を採用します。

### 1. ログイン画面 `/`
* **ヘッダー:** アプリ名ロゴ
* **メイン:** 「だれが学習する？」
  * 登録されているユーザーの「アイコン＋名前」が並んだカードリスト。
  * カードをタップすると、PINコード（4桁の数字キーパッド）を表示。
* **フッター:** なし（シンプルに保つ）

### 2. ダッシュボード `/dashboard`
* **ヘッダー:** 「〇〇さんのトップページ」、今日の連続学習記録（ストリーク）の炎アイコンと日数。
* **メイン:**
  * **[学習をはじめる] セクション**
    * ジャンルの大きなボタン（例: 「算数 ➕」「歴史 🏯」「理科 🔬」などのカード）。
  * **[これまでの記録] セクション**
    * 「今日は〇問といたよ！」「過去のニガテに挑戦（復習）」へのリンクバナー。

### 3. モード・詳細設定画面 `/setup`
* **ヘッダー:** 戻るボタン、「〇〇（ジャンル名）の設定」
* **メイン:**
  * 「どんな問題をやる？」
    * [通常モード]：ランダムに出題
    * [復習モード]：過去に間違えた問題だけを出題
  * 「何問とく？」
    * スライダー（Sliderコンポーネント）またはボタン（5問 / 10問 / 20問など）で選択。
* **ボトムアクション:** 大きな「スタート！」ボタン（プライマリカラー）

### 4. 問題解答画面 `/quiz`
* **ヘッダー:**
  * 左: 戻る（中断する）ボタン
  * 中央: 進捗状況「3問目 / 10問中」のプログレスバー
* **カード（メインエリア）:**
  * **問題文:** 読みやすい大きなフォント（ひらがな多めを想定）。
  * **選択肢:** 2〜4つの選択肢ボタン（縦積み または グリッド）。
* **判定直後のオーバーレイ/ダイアログ:**
  * 正解なら「正解！〇」、不正解なら「残念！✕」のようなエフェクトと音声（将来対応用）。
  * 同時に「解説」のテキスト領域を表示し、「次へ」ボタンで次の問題へ。

### 5. リザルト画面 `/result`
* **ヘッダー:** 「結果」
* **メイン:**
  * 大きくスコア表示「10問中 8問正解！」
  * 称号やメダルアイコンの表示（例: 全問正解で「パーフェクト！」の特別なエフェクト）
  * 解いた問題と、あなたが選んだ答え・正解の振り返りリスト（アコーディオン形式）。
* **ボトムアクション:** 「もう一度おなじジャンルをやる」「ダッシュボードへもどる」ボタン。
