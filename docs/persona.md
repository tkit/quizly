# ペルソナ方針

## 目的
Quizly の文言・UIトーンを一貫させるため、対象ペルソナを明文化する。

## 対象ペルソナ（重要）
- 主対象: 中学年以上（小学校高学年〜中学生）
- 副対象: 保護者（学習履歴の確認・運用）

## 文言トーンの原則
- ひらがな中心の幼い言い回しは避ける
- 簡潔で前向きな表現を使う
- 学習サービスとして自然な語彙を使う（例: 「学習」「選択」「記録」）
- ただし堅すぎる事務文言にはしない

## 問題文・解説文の原則
- 問題文は「〜でしょう？」ではなく「〜ですか？」を基本とする
- 解説の語尾は「〜だよ」ではなく「〜です/〜ます」を基本とする
- 過度に幼い擬音・感嘆表現（例: 「ぐるっと」「天才！」）は避ける
- 正解理由は短く論理的に説明する

## 文言例
- 推奨: 「今日も学習を進めよう」「学習を開始する」「教科選択に戻る」
- 非推奨: 「きょうも がんばろう！」「がくしゅうを おこなう！」「きょうか えらびに もどる」

## 運用ルール
- 新規UI文言を追加する際は、本ドキュメントのトーンに合わせる
- 既存文言を修正する際も、対象年齢が下がる表現になっていないか確認する
- 迷った場合は `docs/requirements.md` のターゲットユーザー定義を優先する
